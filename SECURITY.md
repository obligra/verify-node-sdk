# Security Policy

The Obligra Verify Node.js SDK is currently in early integration preview.

Please do not report security issues through public GitHub issues.

To report a vulnerability or suspected security issue, contact:

security@obligra.ai

Include:

- A clear description of the issue
- Steps to reproduce, if applicable
- Affected package version
- Any relevant logs or error messages with secrets removed

Do not include API keys, secrets, customer data, prompts, model outputs, or production records in any report.

## Supported Versions

| Version | Supported |
|---|---|
| 0.x | Early preview |

Production use requires an approved Obligra Verify account and API credentials.
